# Blinkit Sales Dashboard

## Project Overview
This project analyzes Blinkit sales data using Power BI.

## Features
- Total Sales Analysis
- Outlet Performance
- Item Category Insights
- Fat Content Analysis
- Year-wise Trends
- KPI Cards

## Tools Used
- Power BI
- Excel / CSV
- DAX
- Power Query

## Project Insights
- Tier 3 outlets generated highest sales
- Fruits and Snacks categories performed best
- Regular fat products contributed more revenue
